# 맛집 리뷰 앱

> 맛있는 가게를 발견하고, 리뷰를 작성하고, 다른 사람의 추천을 받는 앱

## 기술 스택
- Next.js 14 (App Router)
- Tailwind CSS + shadcn/ui
- Supabase (인증 + 데이터베이스)
- Vercel 배포

## 시작하기

```bash
npm install
npm run dev
```

## 설계 문서
- `ARCHITECTURE.md` — 페이지 맵, 컴포넌트 계층, 데이터 모델
- `.github/copilot-instructions.md` — AI 코딩 지시사항
