## 수업 목표

AI 코딩 도구(GitHub Copilot)와 협업하여 **실제 동작하는 웹 애플리케이션**을 설계, 구현, 배포할 수 있다.

1. 웹의 동작 원리(HTML, CSS, JavaScript)를 이해하고, AI가 생성한 코드를 읽고 검증할 수 있다
2. Next.js + Tailwind CSS로 프론트엔드를 구성할 수 있다
3. Supabase로 인증, 데이터베이스, 권한 관리를 구현할 수 있다
4. AI 코딩 도구를 효과적으로 활용하고, AI 출력의 오류를 판별할 수 있다
5. Vercel을 통해 매주 결과물을 실제 URL로 배포할 수 있다

## 수업 방식

> **"1주차에 배포하고, 매주 기능을 추가하고, 매주 재배포한다."**

- 매 수업은 **이론 설명 + 실습**으로 진행한다
- 실습은 GitHub Copilot을 활용한 **바이브코딩** 방식으로 한다
- 과제는 **수업 시간 내에 완성하여 제출**한다
- 모든 결과물은 **실제 URL로 배포**하여 제출한다

### AI 도구 사용 원칙

- AI 도구 사용을 **적극 권장**한다
- 단, AI가 생성한 코드를 **이해하고 검증하는 능력**이 핵심이다
- AI 사용 과정을 **로그로 기록**한다 (프롬프트, 오류, 해결 과정)
- AI의 출력을 그대로 제출하는 것이 아니라, **검증하고 수정한 결과**를 제출한다
